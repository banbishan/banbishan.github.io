You can’t change your home page by modifying the file in the public folder

When I first came to hexo, I think I can change my home page by modifying the index.html in the public folder. And I did it. But nothing happened on the web page. Later, I google the mechanism of hexo. The files in the public folder are generated according to the configuration in other files. More details about hexo please refer to [http://www.tuicool.com/articles/fiYVbaY][1]. So if you want change your home page, maybe you should modify the contents in _config.yml, themes folder or other relevant files.
